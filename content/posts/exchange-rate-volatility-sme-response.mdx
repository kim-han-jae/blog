# 환율 변동성, 중소기업은 어디서 타격을 받나

환율 변동의 영향은 수출기업만의 문제가 아닙니다. 원자재·부품·물류비를 수입에 의존하는 내수 기업도 비용 충격을 빠르게 받습니다.

## 우선 대응 순서

1. 외화 노출 구간(매입/매출/차입)을 숫자로 파악
2. 단기 계약과 장기 계약의 환율 반영 조건 재정리
3. 결제 시점 분산과 헤지 도구 적용 검토

## 체크리스트

- [x] 주요 거래처 계약서의 환율 조정 조항을 점검합니다.
- [x] 매입 타이밍을 분산해 평균단가 리스크를 낮춥니다.
- [x] 현금흐름표에 환율 시나리오(±5%, ±10%)를 반영합니다.

## 결론

환율 대응의 핵심은 예측보다 노출 관리입니다. 환율 방향을 맞추려 하기보다, 변동성 자체를 견딜 수 있는 재무구조를 만드는 것이 우선입니다.
